Quais são os tipos de branches?
Feature branch: Criado para desenvolver uma nova funcionalidade
Develop branch: Contém código em nível preparatório para o próximo deploy. Contém várias Features
Bugfix branch: Utilizado para corrigir erros específicos
Release branch: Preparado para a versão final do software
Hotfix branch: Utilizado para correções urgentes em produção
Master branch: Contém código em nível de produção

Offset e Limit
A paginação baseada em offset e limit é utilizada a partir de um deslocamento de registros.
Em outras palavras, você especifica no offset a partir de qual registro você quer os dados, 
e no limit você especifica o limite de registros a serem retornados.
Agora vamos imaginar que você quer 30 vinhos a partir do décimo registro, então você faria a seguinte requisição:
HTTP GET
https://api.minhagastronomia/vinhos?offset=10&limit=30
